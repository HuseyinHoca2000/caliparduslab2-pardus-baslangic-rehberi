# ÇalıPardusLab2 Pardus Başlangıç Rehberi

Bu proje, Pardus işletim sistemine yeni başlayan kullanıcılar için çevrimdışı çalışabilen sade bir başlangıç rehberi geliştirmeyi amaçlamaktadır.

## Proje Amacı

Pardus’a yeni geçen öğrenciler, öğretmenler ve kullanıcılar; temel sistem kullanımı, uygulama yönetimi, ayarlar ve dosya işlemleri gibi konularda rehberliğe ihtiyaç duyabilmektedir. Bu proje ile kullanıcıya adım adım yol gösteren basit ve anlaşılır bir rehber oluşturulması hedeflenmektedir.

## Hedefler

- Yeni kullanıcıların Pardus’a uyum sürecini kolaylaştırmak
- Temel işlemleri sade şekilde anlatmak
- Eğitim kurumlarına uygun rehberlik sunmak
- İnternet olmadan çalışan hafif bir yardımcı araç geliştirmek

## Temel Özellikler

- Başlangıç menüsü
- Temel kullanım rehberi
- Uygulama yönetimi bilgilendirmesi
- Ayarlar ve dosya yönetimi yönlendirmesi
- Offline çalışma

## Hedef Kullanıcılar

- Öğrenciler
- Öğretmenler
- Pardus’a yeni geçen kullanıcılar
- Eğitim kurumları

## Kullanılan Teknolojiler

- Python 3

## Çalıştırma

```bash
python3 src/pardus_baslangic_rehberi.py
